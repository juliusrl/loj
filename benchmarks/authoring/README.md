# ReactDSL Authoring Benchmark Corpus

This directory holds the fixed Stage 2 benchmark corpus for the docs-only authoring lane.

Layout:

- `tasks/`: benchmark task prompts
- `expected/`: deterministic structural checks for each task
- `fixtures/`: sample submission sets for local smoke runs
- `prompts/`: materialized full prompts generated by `rdsl-benchmark export-prompts`

Current task count: `10`

The corpus now includes both:

- broader admin/workflow tasks
- focused `toast` authoring tasks covering string shorthand plus descriptor-shaped `form.*` and `user.*` refs

The baseline lane for Stage 2 is:

- reference docs: `subprojects/rdsl/docs/llm-reference.md`
- task prompt: `tasks/<task-id>.json`
- scoring: compile success + expectation checks + budgets

The harness intentionally scores semantic structure, not exact string output.

Raw model outputs can be normalized into this corpus layout with:

```bash
rdsl-benchmark import --adapter jsonl --input <responses.jsonl> --out-dir <run-dir>
```
