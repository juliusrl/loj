# @loj-lang/rdsl-host-react

Reusable React host helpers for generated frontend-family apps.

Current surface includes:

- shared provider wiring
- Vite host helpers
- exportable React/Vite host template

Local workspace commands:

```bash
npm run build --workspace=@loj-lang/rdsl-host-react
npm run test --workspace=@loj-lang/rdsl-host-react
```
